Vastaukset ensimmäisen teorialuennon kysymyksiin

Q1

Mark these either as True or False:
The number 0 True
The number 1 False
The number 3 False
The number 6 False

Vastaus: 0 True
Kun sijoitellaan, saadaan lopulta 1*(2*(3*(product []))) ja koska product [] = 0,
lopulta kaikki kerrotaan nollalla ja tulos on 0.


Q3
Which of the following are correct and proper uses of if expressions?
(Assume x to be suitably typed variable)

Mark these either as True or False:
if x then "Good" else "Bad" True
if x else "Bad" False
if x then "Good" False
if x == True then "Good" else "Bad" True

Vastaus:
Ensimmäinen ja viimeinen ovat oikein.
If-lausekkeessa pitää olla sekä then että else osa, siksi toinen ja kolmas eivät käy.

Q6
What expression results in the following AST?

Mark these either as True or False:
((map f) x):xs False
(map f) x:xs   False
map f (x:xs)   True
map f x:xs     False

Vastaus: map f (x: xs). Mapin ensimmäinen argumentti on funktio f,
ja sitten tämä funktioiden yhdistelmä ottaa argumentikseen listan (x:xs).


Q7
What is atan2 pi/12 pi*2.1?

(Hint: / and * are left associative, with precedence 7)

Mark these either as True or False:
Function application (f x)  False
Division (E1 / E2)          False
Multiplication (E1 * E2)    False
Syntax error                True

Vastaus: atan2 ottaa kaksi argumenttia, mutta tässä tapauksessa
operaattori / katkaisee argumenttien listan, eli niitä tulee vain yksi,
mitä kääntäjä ei hyväksy.
